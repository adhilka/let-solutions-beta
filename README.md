## 🎓 Courses Offered

Let Solutions delivers practical, lab-oriented training programs designed to foster self-reliance and employability:

### 1. Master Diploma in Chip-Level Engineering (MDCE)
*   **Duration:** 12 Months
*   **Level:** All Levels (Beginner to Expert)
*   **Category:** Laptop & Electronics Chip-Level Repair
*   **Status:** 🔥 Bestseller
*   **Description:** An all-inclusive training program covering diagnostics and chip-level repair of laptops, smartphones, tablets, CCTV systems, and network configurations.

### 2. Junior Penetration Tester (Cybersecurity)
*   **Duration:** 6 Months
*   **Level:** Intermediate
*   **Category:** Cybersecurity & Software
*   **Description:** Dive deep into ethical hacking, security auditing, vulnerability assessments, and web security using industry-standard tools like Nmap, Wireshark, and Metasploit.

### 3. Hardware & Networking Pro
*   **Duration:** 4 Months
*   **Level:** Beginner
*   **Category:** Systems & Infrastructure
*   **Description:** Comprehensive training on computer building, diagnostics, server administration, and complex network setup/troubleshooting.

## 🚀 Application Technical Stack & Features

This application showcases Let Solutions' training offerings, facilitates new admissions, and provides a powerful CMS for management:

1.  **Student Portal:**
    *   Responsive and visually appealing courses browser, detailed syllabus overviews, and student testimonials.
    *   Interactive Admission Form and immediate Enquiry submissions.
2.  **Interactive Blog & Updates:**
    *   Sharing announcements (e.g., new batch alerts, special enrollment discount banners).
3.  **Secure Admin Control Center (`/admin`):**
    *   Update course schemas, admission modes, and inquiry pipelines.
    *   Review and approve user testimonials.
    *   Configure dynamic settings, branding options, announcement banners, and contact information.
4.  **Tech Stack:** 
    *   **Frontend:** React 18, Vite, TypeScript, Tailwind CSS, Lucide icons, Framer Motion for responsive fluid animations.
    *   **Backend:** Express Server (proxying database layers & secure backend routines).
