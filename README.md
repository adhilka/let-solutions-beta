# Let Solutions Technical Institute Application

This is the official web application and administration portal for **Let Solutions Technical Institute**, built with a modern, high-performance web development stack.

---

## 🛠️ Technology Stack

### Frontend Architecture
- **React 18 & TypeScript**: Robust, type-safe interactive components and client-side view management.
- **Vite**: Rapid, optimized bundling and developer asset building.
- **Tailwind CSS**: High-fidelity, custom-crafted utilities for fully responsive visual layouts.
- **Framer Motion**: Smooth, declarative micro-animations and page transitions to enhance the user experience.
- **Lucide React**: Clean, lightweight SVG iconography across all interfaces.

### Backend & Infrastructure
- **Express Server**: Seamless proxying, backend routing, and processing capabilities.
- **Firebase / FireStore**: Secure cloud persistence, real-time database structures, and strict security rule declarations.

---

## 🚀 Technical Core Directory Structure
- `/src/pages`: Visual interfaces containing Student portals, admissions forms, courses catalog, and robust secure administration dashboards (`/admin/*`).
- `/src/components`: Reusable user-interface elements, including dynamic navigation systems, cards, and layouts.
- `/src/constants`: Failsafe mock and backup structures, along with system configuration fields.
- `/src/types`: Centralized TypeScript module definitions and schemas.
